

# Smart Utility Toolkit

A modern multi-tool Flutter mobile application that combines everyday utilities like unit conversion, notes, BMI calculator, and task management into one clean, offline-first app.

---

##  Features

###  Home Dashboard

* Lottie animations for modern UI feel

###  Unit Converter

* Convert common units (length, weight, temperature)
* Fast and simple interface

###  Notes Manager

* Create, edit, and delete notes
* Offline storage using local device memory
* Animated empty state (Lottie)

###  Task Manager

* Create tasks
* Mark tasks as completed
* Edit and delete tasks
* local storage (offline-first)

###  BMI Calculator

* Calculate Body Mass Index
* Instant category feedback (Underweight, Normal, Overweight)

---

## Offline Support

This app works **100% offline** using local storage:

* Uses `shared_preferences`
* No internet required
* Data is saved directly on device

---

## Tech Stack

* Flutter (Dart)
* Material 3 UI
* Shared Preferences (local storage)
* Lottie Animations
* Solar Icons

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Faithy-dev/smart-utility-toolkit.git
```

### 2. Go into project folder

```bash
cd smart-utility-toolkit
```

### 3. Install dependencies

```bash
flutter pub get
```

### 4. Run the app

```bash
flutter run
```

---

##  Build APK

To generate a release APK:

```bash
flutter build apk --release
```

APK output:

```
build/app/outputs/flutter-apk/app-release.apk
```
